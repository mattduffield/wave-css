wc-field - Read-Only Informational Component

The wc-field component is a read-only, informational display component that is NOT part of form submissions. It's perfect for displaying data with optional labels and links.

Usage Instructions

Basic Syntax:

<wc-field label="Label Text" value="Display Value"></wc-field>

With Link:

<wc-field
  label="Bill Plans"
  link="/v/bill_plans/create"
  value="3 plans available">
</wc-field>

With HTMX:

<wc-field
  label="Bill Plans"
  link="/v/bill_plans/create"
  hx-get="/v/bill_plans/create?script_id=123"
  hx-target="#modal-container"
  hx-swap="innerHTML"
  hx-push-url="false"
  value="Manage plans">
</wc-field>

Complex Content (No Slots - Just Child Elements):

<wc-field label="Bill Plans">
  <div class="text-center text-xs text-4">
    {{Record.data.bill_plan}}
  </div>
</wc-field>

Available Attributes:

  • label - Label text (optional)
  • label-class - Custom CSS classes for label
  • value - Display value (if not using child content)
  • value-class - Custom CSS classes for value area
  • link - URL for clickable label
  • text-align - Text alignment (left, center, right)
  • hx-* - Any HTMX attributes (hx-get, hx-target, etc.)

Key Features:

  • ✅ Read-only informational component
  • ✅ Does NOT participate in form submission
  • ✅ Optional label (only renders if provided)
  • ✅ Supports both simple values and complex child content
  • ✅ Clickable labels with link support
  • ✅ Full HTMX integration
  • ✅ Customizable styling via classes
  • ✅ No Shadow DOM (follows Wave CSS patterns)
