Color Variation Modes

The color-variation attribute controls the color diversity:

  • standard - Uses --surface-4, --surface-6, --surface-7, --surface-8, --surface-10 (strong visible differences)
  • subtle - Uses HSL lightness adjustments (-20%, -10%, 0%, +10%, +20%, +30%)
  • off - All bars/segments same color

Default: standard for chart-bar, chart-pie, and chart-doughnut, off for others

Custom Color Levels

Use the color-levels attribute to specify which surface variables to use in standard mode.

Default Levels (4,6,7,8,10)

color-variation="standard"

Custom Levels (3,5,7,9,11)

color-levels="3,5,7,9,11"

Code Examples

Basic Usage

<!-- Simple usage -->
<wc-busy-indicator type="chart-bar"></wc-busy-indicator>

<!-- With text and size -->
<wc-busy-indicator
  type="chart-line"
  text="Loading data..."
  size="large">
</wc-busy-indicator>

<!-- With color variations -->
<wc-busy-indicator
  type="chart-bar"
  color-variation="standard"
  size="medium">
</wc-busy-indicator>

<!-- Subtle color variations -->
<wc-busy-indicator
  type="chart-bar"
  color-variation="subtle">
</wc-busy-indicator>

<!-- No color variations -->
<wc-busy-indicator
  type="chart-bar"
  color-variation="off">
</wc-busy-indicator>

<!-- Custom color levels -->
<wc-busy-indicator
  type="chart-bar"
  color-variation="standard"
  color-levels="4,6,7,8,10">
</wc-busy-indicator>

With WC-ChartJS

<!-- Automatic busy indicator (matches chart type) -->
<wc-chartjs
  url="/api/data"
  type="bar"
  busy-indicator="true">
</wc-chartjs>

<!-- Custom busy indicator type -->
<wc-chartjs
  url="/api/data"
  type="line"
  busy-indicator="true"
  busy-indicator-type="chart-ecg">
</wc-chartjs>

Programmatic Control

<wc-busy-indicator id="myIndicator" type="spinner"></wc-busy-indicator>

<script>
  const indicator = document.getElementById('myIndicator');

  // Show the indicator
  indicator.show();

  // Hide the indicator
  indicator.hide();

  // Remove from DOM
  indicator.remove();

  // Change type dynamically
  indicator.setAttribute('type', 'chart-bar');

  // Change color variation mode
  indicator.setAttribute('color-variation', 'standard'); // or 'subtle' or 'off'
</script>

Attributes Reference

Attribute Type Default Description
type string spinner Animation type: chart-bar, chart-line, chart-ecg, horizontal-bar, chart-connector, chart-pie, chart-doughnut, spinner, pulse, dots, skeleton
text string "" Optional text to display below the indicator
size string medium Size: small, medium, large
color-variation string standard* / off Color variation mode: "standard" (--surface-x), "subtle" (HSL adjustments), "off" (single color). *Defaults to "standard" for chart-bar, chart-pie, and chart-doughnut, "off" for others
color-levels string "4,6,7,8,10" Comma-separated surface variable levels to use in "standard" mode (e.g., "3,5,7,9,11"). Only applies when color-variation is "standard".
color string theme color Custom color override (CSS color value)
