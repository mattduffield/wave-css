wc-help-drawer

Demo

Click the "?" button or press Ctrl + / to open the help drawer.

Trigger button: 

      


      

        
Features

        
    
          
  • Circular "?" trigger button (inverts on hover)
    • 
          
  • Fixed right slide-out panel (420px) with overlay
    • 
          
  • Three tabs: Help, Create Ticket, My Tickets
    • 
          
  • Help tab: search with debounced HTMX content loading
    • 
          
  • Create Ticket: type, priority, title, description + screenshot capture
    • 
          
  • My Tickets: fetches and renders ticket cards with status badges
    • 
          
  • Keyboard: Ctrl+/ to toggle, Escape to close
    • 
          
  • Mobile responsive (full width under 480px)
    • 
        

      


      

        
Public API

        

          drawer.open() — Open the drawer

          drawer.close() — Close the drawer

          drawer.toggle() — Toggle open/close
        

        

          
          
          
        

      


      

        
Usage

        
<wc-help-drawer
  reference-key="DS-COMPARE-001"

Events

        

          wchelpdraweropen — Fired when drawer opens

          wchelpdrawerclose — Fired when drawer closes

          wchelpticketcreated — Fired after ticket submission (detail: { ticketId })
        

      


    

  


  
  
    var drawer = document.getElementById('demo-drawer');
    var log = document.getElementById('drawer-log');
    if (drawer && log) {
      drawer.addEventListener('wchelpdraweropen', function() {
        log.textContent = 'Event: wchelpdraweropen';
      });
      drawer.addEventListener('wchelpdrawerclose', function() {
        log.textContent = 'Event: wchelpdrawerclose';
      });
      drawer.addEventListener('wchelpticketcreated', function(e) {
        log.textContent = 'Event: wchelpticketcreated | ticketId: ' + (e.detail.ticketId || 'n/a');
      });
    }
