wc-wizard

Static Wizard Demo

This demo uses inline content to test the wizard UI without a server. Step content is injected via JavaScript.


      


      
      

        
Public API

        

          wizard.next() — Go to next step

          wizard.back() — Go to previous step

          wizard.goTo(n) — Go to step by index (0-based)

          wizard.getCurrentStep() — Current step index

          wizard.getTotalSteps() — Total number of steps

          wizard.isFirst() / wizard.isLast() — Position checks
        

      


      
      

        
HTMX Usage

        
<wc-wizard id="my-wizard" form-id="my-form">
  <wc-wizard-step label="Basics"
    hx-get="/api/step1"
    hx-include="#my-form"
    hx-indicator="#loader">
  </wc-wizard-step>
  <wc-wizard-step label="Details"
    hx-get="/api/step2"
    hx-include="#my-form"
    before-navigate="saveState">
  </wc-wizard-step>
</wc-wizard>
