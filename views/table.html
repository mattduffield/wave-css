wc-table

A lightweight data-driven table component. Define columns with <wc-table-col>, load data from a URL or static JSON. Uses the existing wc-table CSS classes internally.

Static Data with Column Definitions

Data provided via items attribute. Columns defined with <wc-table-col>. Click headers to sort.

<wc-table striped hoverable clickable items='[...]'>
  <wc-table-col field="first_name" label="First Name" sortable></wc-table-col>
  <wc-table-col field="deal_value" label="Deal Value" align="right" format="currency" sortable></wc-table-col>
</wc-table>

Compact Table (size="sm")

Dense data view with smaller padding. Good for dashboards and data exploration.

Auto-Columns from display-member

No <wc-table-col> needed. Just list field names in the display-member attribute.

<wc-table striped hoverable
          display-member="name,email,phone,city"
          items='[...]'>
</wc-table>

Empty State

Custom message when no data is available.

Date and Number Formatting

Use format attribute on columns: date, datetime, currency, number, boolean.

Programmatic Data Update

Set data via JavaScript using the .data property or .refresh() method.

// Row click events const clickTable = document.getElementById('click-table'); const clickOutput = document.getElementById('click-output'); if (clickTable && clickOutput) { clickTable.addEventListener('wctablerowclick', (e) => { clickOutput.innerHTML = 'Click: ' + JSON.stringify(e.detail.data); }); clickTable.addEventListener('wctablerowdblclick', (e) => { clickOutput.innerHTML = 'Double-click: ' + e.detail.data.name + ' (' + e.detail.data.industry + ')'; }); } // Programmatic data window.loadSampleData = function() { const table = document.getElementById('programmatic-table'); table.data = [ { name: 'Alice Chen', role: 'Engineer', department: 'Engineering' }, { name: 'Bob Martinez', role: 'Designer', department: 'Product' }, { name: 'Carol Kim', role: 'Manager', department: 'Engineering' }, { name: 'Dave Patel', role: 'Analyst', department: 'Data' }, { name: 'Eve Thompson', role: 'Director', department: 'Operations' } ]; }; window.clearData = function() { const table = document.getElementById('programmatic-table'); table.data = []; };
