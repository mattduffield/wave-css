Keyboard Navigation

Click the divider to focus it, then use:

  • Arrow Left/Right — resize horizontal split (10px steps)
  • Arrow Up/Down — resize vertical split (10px steps)
  • Shift + Arrow — resize in 50px steps
  • Home — shrink to minimum size
  • End — expand to maximum size
  • Enter — toggle collapse (if collapsible)
