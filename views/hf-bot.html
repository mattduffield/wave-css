Wave CSS Hugging Face Bot Component

Overview

The HF Bot component provides an interactive chat interface powered by Hugging Face's Transformers.js, running AI models directly in the browser using WebGPU acceleration.

Note: Like WebLLM, models are downloaded to the browser (1-2GB) and cached for subsequent use.

Key Differences from wc-ai-bot:

  • Uses Hugging Face ecosystem and model hub
  • Different model format (ONNX/GGUF vs MLC)
  • Simpler API but similar performance
  • Same GPU requirements and limitations

System Requirements: Same as wc-ai-bot - requires capable GPU and 4GB+ available RAM.

Basic Usage

Click the chat bubble in the bottom-right corner to interact with the Hugging Face assistant:

<wc-hf-bot 
  bot-id="demo-hf-basic"
  title="HF Assistant"
  placeholder="Ask me anything..."
  system-prompt="You are a helpful AI assistant powered by Hugging Face.">
</wc-hf-bot>

Embedded Bot (Minimal Theme)

The minimal theme for inline chat interfaces:

<div style="height: 500px;">
  <wc-hf-bot 
    bot-id="demo-hf-minimal"
    theme="minimal"
    title="HF Chat"
    model="onnx-community/Llama-3.2-1B-Instruct-q4f16_1_gguf">
  </wc-hf-bot>
</div>

Available Models

Transformers.js supports various ONNX and GGUF format models:

TinyLlama 1.1B

Very small and fast (~550MB)

model="Xenova/TinyLlama-1.1B-Chat-v1.0"

Phi-3 Mini

Microsoft's efficient model (~2.5GB)

model="Xenova/Phi-3-mini-4k-instruct"

DistilGPT2

Tiny model for testing (~250MB)

model="Xenova/distilgpt2"

GPT-2

Classic model (~500MB)

model="Xenova/gpt2"

Comparison: wc-ai-bot vs wc-hf-bot

Feature wc-ai-bot (WebLLM) wc-hf-bot (Transformers.js)
Library WebLLM (MLC) Transformers.js (Hugging Face)
Model Format MLC compiled ONNX/GGUF
Model Hub Limited selection Hugging Face Hub
Performance Slightly faster Similar
Library Size ~1MB ~500KB
GPU Requirements Same Same

JavaScript API

Same API as wc-ai-bot for consistency:

// Get bot reference
const bot = document.querySelector('[bot-id="demo-hf-basic"]');

// Send a message
bot.sendMessage("Hello!");

// Clear conversation
bot.clearConversation();

// Export conversation
const history = bot.exportConversation();

// Update system context
bot.setContext("You are an expert in JavaScript.");

// Check system support
const support = await WcHfBot.checkSystemSupport();
console.log('GPU:', support.renderer);
console.log('Supported:', support.supported);

// Get available models
const models = await WcHfBot.getAvailableModels();
console.table(models);

All Available Attributes

Same attributes as wc-ai-bot for consistency:

Core Attributes

bot-id
Unique identifier for the bot instance
model
Hugging Face model ID to use
system-prompt
Instructions for the AI's behavior

UI & Safety

theme
"bubble" (default), "minimal", or "sidebar"
check-gpu-compatibility
"true" (default) or "false"
force-enable
"true" to bypass GPU checks
