Context Menu

A reusable context menu component. Supports declarative child items or programmatic WcContextMenu.show(x, y, items) calls. Only one menu is visible at a time.

Programmatic (right-click the box)

Right-click anywhere in this box

Declarative (right-click the box)

Right-click for declarative menu

Button Triggered

Click the button to open a context menu at the button's position.

API Reference

// Programmatic — show at x/y with items array
WcContextMenu.show(event.clientX, event.clientY, [
  { label: 'Query', icon: 'magnifying-glass', action: () => doQuery() },
  { label: 'View Indexes', icon: 'bolt', action: () => viewIndexes() },
  { divider: true },
  { label: 'Import Data', icon: 'file-import', action: () => importData(), disabled: true },
]);

// Hide programmatically
WcContextMenu.hide();
