Commands

  • /create-schema — LiteSpec schema
  • /create-list — List template (fast)
  • /create-edit — Edit template (fast)
  • /create-template — List or edit (slower)
  • /create-screen — Full screen
  • /create-component — Wave CSS component
  • /create-web-pilot — Playwright script
  • /help — Show all commands

Example Prompts

  • /create-schema contact with name, email, phone, status
  • /create-list article with title, category, release date
  • /create-edit article with title, description, category
  • How do field rules work?

Notes

  • First load downloads model (~1-2 GB)
  • Commands select relevant KB sections to fit context
  • General questions use keyword matching
  • Set debug="true" to see context injection in console
