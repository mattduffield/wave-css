Wave CSS AI Bot Component

Overview

The AI Bot component provides an interactive chat interface powered by WebLLM, running AI models directly in the browser. It supports multiple themes, customizable models, and seamless integration with your application.

Note: The first time a model loads, WebLLM will download it (typically 1-2GB). The model is cached in the browser for subsequent uses.

Download Tips:

  • Downloads may take 5-15 minutes depending on your connection
  • If download stalls, try refreshing the page
  • Chrome/Edge typically work best for large downloads
  • Keep the browser tab active during download

Basic Usage

Click the chat bubble in the bottom-right corner to interact with the AI assistant:

<wc-ai-bot 
  bot-id="demo-basic"
  title="AI Assistant"
  placeholder="Ask me anything..."
  system-prompt="You are a helpful AI assistant. Be concise and friendly in your responses.">
</wc-ai-bot>

Embedded Bot (Minimal Theme)

The minimal theme is perfect for embedding the chat interface inline:

<div style="height: 500px;">
  <wc-ai-bot 
    bot-id="demo-minimal"
    theme="minimal"
    title="Embedded Chat"
    model="Llama-3.2-1B-Instruct-q4f32_1-MLC">
  </wc-ai-bot>
</div>

JavaScript API

Control the bot programmatically:

// Get bot reference
const bot = document.querySelector('[bot-id="demo-basic"]');

// Send a message
bot.sendMessage("Hello!");

// Clear conversation
bot.clearConversation();

// Export conversation
const history = bot.exportConversation();

// Update system context
bot.setContext("You are an expert in JavaScript.");

// Toggle minimize (bubble theme)
bot.toggleMinimize();

Events

Listen to bot events:

// Bot ready
document.addEventListener('bot:ready', (e) => {
  console.log('Bot ready:', e.detail.botId, e.detail.model);
});

// Message sent
document.addEventListener('bot:message-sent', (e) => {
  console.log('User:', e.detail.message);
});

// Response received
document.addEventListener('bot:response-received', (e) => {
  console.log('Bot:', e.detail.response);
});

// Error handling
document.addEventListener('bot:error', (e) => {
  console.error('Error:', e.detail.error);
});

Complete Example

<!-- Auto-opening support bot with custom styling -->
<wc-ai-bot 
  bot-id="support"
  title="Support Assistant"
  theme="bubble"
  position="bottom-right"
  auto-open="true"
  model="Llama-3.2-3B-Instruct-q4f32_1-MLC"
  temperature="0.5"
  max-tokens="2000"
  placeholder="How can I help you today?"
  system-prompt="You are a helpful support agent for a software company. 
    Be professional, concise, and friendly. Focus on solving user problems.">
</wc-ai-bot>

<script>
  // Listen for when bot is ready
  document.addEventListener('bot:ready', (e) => {
    if (e.detail.botId === 'support') {
      console.log('Support bot is ready!');
      
      // Send a welcome message after a delay
      setTimeout(() => {
        const bot = document.querySelector('[bot-id="support"]');
        bot.sendMessage('Welcome! How can I assist you today?');
      }, 2000);
    }
  });
  
  // Track user interactions
  document.addEventListener('bot:message-sent', (e) => {
    if (e.detail.botId === 'support') {
      // Analytics tracking
      console.log('User query:', e.detail.message);
    }
  });
</script>
