Pivot Table Demo

The wc-pivot component renders cross-tabulation pivot tables from raw JSON data with auto-detection, four-zone field panel, value filters, heatmap, sorting, and CSV export.

Field Panel: Click ☰ or use show-field-panel

The field panel shows all available fields with type badges. Click a field to assign it to Rows, Columns, Values, or Filters. Click the ⧫ icon on a chip to filter values. Click the value chip label (e.g., "Sum of Amount") to change the aggregate function.

Auto-Detect: Prospects by State and Status (Count)

Auto-detects row/column fields and defaults to count aggregation. Click ☰ to open the field panel.

Pre-Configured: Revenue by State and Status (Sum, Currency)

Same data as above but configured to sum the amount field with currency formatting.

Boolean Column: Users by Role and Active Status (Average Score)

Boolean fields are natural 2-column pivots. Shows average score with decimal format.

Sales: Product × Quarter (No Controls)

Embedded dashboard style — controls hidden, heatmap on.

Date Grouping: Sales Over Time

Date fields are now detected and can be grouped. Open the field panel, click the 📅 icon on the date chip to change grouping (Year, Quarter, Month). Try the compact layout toggle (▤/▦) in the controls bar.

Datetime: Server Logs by Timestamp and Severity

Data with real timestamps — defaults to Month grouping. Click 📅 on the date chip to switch between Day, Datetime, Quarter, etc. Try Datetime to see exact times, or Day to ignore time.

Config Save / Load

Configure the pivot above, then click "Save Config" to capture the current state. Click "Load Config" to restore it. Uses getConfig() and loadConfig() methods.

Dynamic Data Update

Click buttons to load different datasets. The pivot re-analyzes and re-renders.

Drill-Down: Click Any Cell

Click any cell to expand an inline detail table showing the matching source documents. Click the same cell again (or ×) to collapse. The pivot:drill-down event fires with the documents.
