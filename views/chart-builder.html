Chart Builder Demo

The wc-chart-builder component renders interactive charts from raw JSON data with auto-detection and configurable field pickers.

Auto-Detect: Aggregation Results (Bar)

Grouped by _id with a count field — auto-detects as a bar chart.

Auto-Detect: Few Categories (Pie)

With fewer than 8 categories and one value field, auto-detects as a pie chart.

Auto-Detect: Time Series (Line)

When the label field contains ISO dates, auto-detects as a line chart.

Multi-Series Bar: Sales by Region (3 Product Lines)

Three value fields create a grouped bar chart. Use the controls to add/remove series or switch to line.

Multi-Series Line: Monthly Metrics (12 Months, 4 Series)

Multiple value fields over time. Try switching to bar, area, or removing series.

Number/KPI Display

Single value displayed prominently. Set chart-type="number".

Pre-Configured: Multi-Series (No Controls)

Revenue vs Cost line chart with controls hidden.

Doughnut Chart

Explicitly set chart-type="doughnut".

Area Chart

Filled line chart with chart-type="area".

Dynamic Data Update

Click buttons to swap data — chart re-renders with new schema and values.
