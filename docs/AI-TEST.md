# AI Test


Command Examples

## Create Schema

```bash
/create-schema article with title, description, release date, category

/create-schema invoice with customer name, email, billing address, line items array, total, status, due date

/create-schema employee with first name, last name, email, department, hire date, salary, is active
```

## Create List

```bash
/create-list article with columns for title, category, release date, is active

/create-list invoice with columns for customer name, total, status, due date, modified date

/create-list employee with columns for last name, first name, department, email, is active
```

## Create Edit

```bash
/create-edit article with title, description, release date, category

/create-edit invoice with customer name, email, line items, total, status, due date, notes

/create-edit employee with first name, last name, email, department, hire date, salary
```

## Create Template

```bash
/create-template article list and edit templates with title, description, category, release date
```

## Create Screen

```bash
/create-screen ticket tracker with title, description, priority, assignee, status, due date
```

## Create Component

```bash
/create-component progress bar with percentage, color, animated fill, and label

/create-component star rating with max stars, value, and readonly mode
```

## Create Web Pilot

```bash
/create-web-pilot automate SafeCo auto quote for NC using prospect data
```
